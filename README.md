# AI Trainer Portfolio

## 简介
这是一个为 AI 训练师/AI 产品经理设计的个人作品集网站。包含首页、AI 资讯、工具栈展示及求职意向板块。

## 本地运行
1.  下载代码
2.  在项目目录下运行 `python -m http.server 8080` (如果你安装了 Python)
3.  在浏览器访问 `http://localhost:8080`

## 部署到 GitHub Pages
1.  在 GitHub 上创建一个新的仓库 (Repository)。
2.  将本地代码推送到 GitHub:
    ```bash
    git remote add origin <你的 GitHub 仓库地址>
    git branch -M main
    git push -u origin main
    ```
3.  在 GitHub 仓库设置 (Settings) -> Pages 中，选择 `main` 分支作为 Source，点击 Save。
4.  你的网站将会被部署在 `https://<你的用户名>.github.io/<仓库名>/`。
